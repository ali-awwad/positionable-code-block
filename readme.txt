== Description ==

The Positionable Code Block registers single block that is to use, it allows you to add html, style, and script tags to your pages.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mytest` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

You can also install using composer. Check [View on GitHub](https://github.com/ali-awwad/positionable-code-block) for documentation on how to install using composer.
